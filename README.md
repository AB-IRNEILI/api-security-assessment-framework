# 🛡️ REST API Security Assessment Framework in Microservices

## 📝 Project Overview
This repository contains the documentation and architectural overview of a security assessment framework specifically designed for REST APIs within microservices environments, tailored for the financial sector. 

*Note: The source code of this project is kept in a private repository for security and data confidentiality reasons.*

## 🚀 Key Features
- **Automated Vulnerability Scanning:** Tailored checks focusing on the OWASP Top 10 vulnerabilities.
- **Microservices Compatibility:** Designed to seamlessly analyze APIs operating in containerized ecosystems.
- **Financial Sector Standards:** Implements specific security baseline assessments required for banking environments.

## 🛠️ Architecture & Tech Stack
- **Backend & Core Logic:** Python
- **Deployment & Containerization:** Docker
